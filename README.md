# 🎓 Student Performance Analysis Using Machine Learning

An end-to-end Data Science project built to analyze, predict, and cluster student academic performance using Python.

## 📊 Project Summary
- **Regression:** Predicted final exam scores using Linear, Ridge, and Lasso Regression. **Lasso** achieved the best performance with the lowest RMSE.
- **Clustering:** Segmented students into **4 distinct profiles** using the **K-Means** algorithm, optimized via the **Elbow Method**.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `seaborn`

## 📁 Repository Structure
- `Student-Performance-Analysis-Report.docx`: The full technical report with detailed visualizations.
- `student_performance_analysis.ipynb`: Jupyter Notebook containing the full source code.
- `README.md`: Project documentation.

## 🚀 How to Run
1. Clone the repository.
2. Install requirements: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Open and run `student_performance_analysis.ipynb`.
